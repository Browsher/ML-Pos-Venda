# Template: Compra Confirmada

Use este template para gerar a mensagem quando o comprador finaliza um pedido.

## Dados disponíveis
- Nome do comprador
- Nome do produto comprado
- Número do pedido

## Tom
- Caloroso e profissional
- Confirmar que o pedido foi recebido
- Informar que será notificado quando enviar
- Curto (3-4 linhas máximo)

## Exemplo de saída esperada
Olá [Nome]! Seu pedido de [Produto] foi confirmado com sucesso. Já estamos separando e em breve você receberá o código de rastreamento. Qualquer dúvida, é só chamar!
